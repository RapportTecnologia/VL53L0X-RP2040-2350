Descrição detalhada

Interface C++ para o sensor de distância VL53L0X (Time-of-Flight) no RP2040.

Esta classe provê uma API de alto nível para inicialização, configuração e leitura de distância do sensor VL53L0X através do barramento I2C. O objetivo é facilitar a integração em aplicações RP2040 (Raspberry Pi Pico/Pico W) mantendo compatibilidade com padrões comuns.
