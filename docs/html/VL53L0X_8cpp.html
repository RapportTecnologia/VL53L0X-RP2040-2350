Descrição detalhada

Implementação da classe VL53L0X para RP2040.

A maior parte da funcionalidade desta biblioteca é baseada na API do VL53L0X fornecida pela ST (STSW-IMG005). Alguns comentários explicativos foram adaptados do código-fonte da API, do manual do usuário (UM2039) e do datasheet do VL53L0X.
